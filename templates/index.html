This is a Stroke Prediction Dashboard.

Dataset source on Kaggle

The features of this app allow you to explore the Stroke Prediction dataset. Use the custom querying feature or visualise the charts for age, gender, hypertension based data.

  • Gender - View the stroke cases by gender.
  • Age Distribution - Stroke cases by age groups.
  • Hypertension - Correlation between hypertension and stroke prediction.
  • Custom Query - Create your own SQL queries.
